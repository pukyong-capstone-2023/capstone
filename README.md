
## 필요한 의존성

- JDK 17
- Npm

## 실행

1. 빌드 파일 실행

```shell
./gradlew build
```

2. 애플리케이션 실행

```shell
java -jar ./build/libs/capstone-*-SNAPSHOT.jar
```

3. 랜딩 페이지 접속

```http request
http://localhost:8080
```

## 정보

프론트 패키지 위치

```shell
cd ./src/main/frontend
```

백엔드 패키지 위치

```shell
cd ./src/main/java
```
